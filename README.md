# Agent-Based Call Drop Analysis

This project uses an AI Agent with Retrieval-Augmented Generation (RAG)
to analyze telecom call drop issues and recommend solutions.

## Tech Stack
- Python
- LangChain
- Mistral-7B (Open Source)
- ChromaDB
- SentenceTransformers

## How to Run
1. python rag/ingest.py
2. python main.py
